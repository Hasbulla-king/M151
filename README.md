# M151

Damit man eine Entwicklungsumgebung mit einem Webserver benützen kann muss man diese zuerst installieren.

Dafür kann man im Internet "Xampp" eingeben und den Webserver installieren.

<img width="577" height="449" alt="image" src="https://github.com/user-attachments/assets/deebec4b-807c-4b7e-abad-24bee1982fb8" />

Wenn man den Webserver installiert hat, kann man den Apache Service starten.

<img width="493" height="321" alt="image" src="https://github.com/user-attachments/assets/cd4253df-e4a3-4f6c-9f86-b9722fb86ce5" />

Als Entwicklungsumgebung kann man VS Code herunterladen. Dafür kann man VS Code im Internet eingeben und beim ersten Link dann auf "herunterladen" klicken.

<img width="818" height="523" alt="image" src="https://github.com/user-attachments/assets/503f6827-8f32-44f5-b394-b52b599f9c6d" />

Danach kann man mit dem Programmieren anfangen.
